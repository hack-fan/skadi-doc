## 2021年4月10日 初始版本

把这个版本当作初始版本吧，目前的功能有：

* 通过公众号进行 Agent 管理
* 通过公众号给 Agent 发布任务
* Agent 任务相关所有接口
* Agent [Golang SDK](https://github.com/hack-fan/skadigo)
* 一个通用版的 [Agent](https://github.com/hack-fan/skadi-agent-shell)
* 用户发送公众号通知相关接口
* Agent 发送通知相关接口
