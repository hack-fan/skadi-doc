HTTP API 文档在顶端目录里边。[点击这里](/ref/)也可以直接访问。

API 分为 User 系列和 Agent 系列。

* User 系列使用用户的 User Token，一般用在 CI 等主动发起场合。
* Agent 系列使用 Agent Token，可以用来写 Agent。

然后为了方便大家使用，会陆续推出各个语言的 SDK，会尽量都写成0依赖的。

这个章节就是各 SDK 的简单说明。
