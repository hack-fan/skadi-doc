{{< /blocks/cover>}} {{% blocks/lead color="primary" %}} 大家好，这个工具是我为个人开发者和小团队开发的，我自己就是第一个用户。 我们一般会有数量不多的一些云服务器和线上服务。 有时候不在电脑边，想查看一下服务状态或者需要重启某个服务； 有时候不太放心把私钥交给CI系统，又没有用来部署的API，只能手动重部署； 有时候会有一点管理需求，但是做一个前端控制台又不划算（其实是不会）。 所以我做了这个小服务，让我们可以把微信公众号当做一个控制台随手使用。 {{% /blocks/lead %}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fa-lightbulb" title="简单" %}} 关注 LetServerRun，根据引导在云服务器复制粘贴几行命令就可以开始使用。 {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-shield-alt" title="安全" %}} 云端服务不会直接控制你的服务器，而只是暂存你的命令和消息等待服务器拉取。 {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="开源" %}} 开源项目是方便你审查所运行的代码，也为专业用户自己编写 Agent 提供参考。 {{% /blocks/feature %}} {{< /blocks/section>}}
